Green One

As of June 29, 2015, the game can be found here (currently hosted on Joel's now-defunct student website)

Challenge yourself to cross side-scroller worlds in order to save the galaxy! In Green One, you must utilize your Gravity Switch and Laser Cannon to make your way across dangerous planets.

Green One was built as a team project by University of Washington students for Computational Worlds. We incorporated the Phaser.io framework, and it uses a combination HTML5/CSS, JavaScript, TypeScript, and JSON to achieve amazing results.

We have incorporated elements of agent-oriented and object-oriented programming. If desired, you can create your own levels and implement them using the Level0 superclass. New Enemies can be crafted. New PowerUps can be designed. All using already implemented code!

Lovingly made by Joel Hunsaker, Aaron Schraufnagel, and Kyle Sessions, with help from Marilyn Ferguson.
